# Parallax-Animation-UI-Web-Design
This project focuses purely on highlighting smooth parallax scrolling and interaction in web design — keeping the design minimal and motion-focused.
✨ What I used in this project:
• Background images from Lummi
• Icons via Iconify plugin
• Parallax effect built using Figma Prototyping
• Animations with Smart Animate, After Delay, On Click
• Custom timing curves for smooth motion flow
The main goal was to create a clean visual experience where motion brings the interface to life.
